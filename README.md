# Termux Parrot OS Shell
### Make some different look of your termux

## Installation
+ Copy commands and paste it in Your Termux

+ ```pkg update && pkg upgrade```
+ ```pkg install git -y```
+ ```git clone https://github.com/Ptechgithub/BannerX.git```
+ ```cd BannerX```
+ ```chmod +x *```
+ ```bash setup.sh```
+ ```exit```
